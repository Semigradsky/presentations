Cosine

			cos(cos(cos(0))) -> 0.857553...
			...
			cos(cos(...cos(0)...)) -> 0.739085...

Fixed point

x = f(x) = f(f(x)) = f(f(f(x))) = ...
			f(x) = cos(x) -> fixpoint = 0.739085...
			f(x) = x2 - 3x + 4 -> fixpoint = 2
			f(x) = x2 -> fixpoint = 0, fixpoint = 1
			f(x) = x + 1 -> no fixpoint

Factorial

				function factorial(n) {
					return n === 0 ? 1 : n * factorial(n-1);
				}

Non-recursive factorial

				function almostFactorial(f) {
					return function(n) {
						return n === 0 ? 1 : n * f(n-1);
					}
				}

Fixed point

factorial
is a fixed point of 
almostFactorial
!

				almostFactorial(factorial)(4) === factorial(4); 
				almostFactorial(factorial)(10) === factorial(10);
				almostFactorial(factorial)(17) === factorial(17);
				...
				But we are trying to calculate factorial without recursion!

Y comes to help!

What if there exists function 
Y
that we could pass 
almostFactorial
to that would return the fixed point of it, i.e. 
factorial
function?

			Y(almostFactorial)(42) =>
			factorial(42) =>
			almostFactorial(factorial)(42) =>
			almostFactorial(Y(almostFactorial))(42)
			So we do not need factorial function at all!
