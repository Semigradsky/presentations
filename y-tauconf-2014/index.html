Fixed point

x = f(x) = f(f(x)) = f(f(f(x))) = ...
			f(x) = cos(x) -> fixpoint = 0.739085...
			f(x) = x2 - 3x + 4 -> fixpoint = 2
			f(x) = x2 -> fixpoint = 0, fixpoint = 1
			f(x) = x + 1 -> no fixpoint
