λ-calculus

λ-expressions

    t ::= terms:
  • x variable
  • λx.t abstraction
  • t t application
    v ::= values:
  • λx.t abstraction value

Simple λ functions

  • λx.x identity
  • λs.s s self-application
  • λf.λx.f x function application

Free variables

  1. x in x
  2. x in λx1.t if
    • x is not x1 and
    • x is free in t
  3. x in (t1 t2) if
    • x is free in t1 or
    • x is free in t2

Bound variables

  1. x in λx1.t if
    • x is x1 or
    • x is bound in t
  2. x in (t1 t2) if
    • x is bound in t1 or
    • x is bound in t2

Free and bound variables

λfunc. λarg.(func arg)

λx.λy.(x (y λx.x))

Normal order β reduction

(<function> <argument>)

  1. normal order β reduce <function> to <function value>
  2. if <function value> is λ<name>.<body> then replace all free occurrences of <name> in <body> with <argument> and normal order β reduce the new <body>
  3. if <function value> is not a function then normal order β reduce <argument> to <argument value> and return (<function value> <argument value>)

Normal order β reduction

  • ((λfunc.λarg.(func arg) λx.x) λs.(s s)) =>
  • (λarg.(λx.x arg) λs.(s s)) =>
  • (λx.x λs.(s s)) =>
  • λs.(s s)

Applicative order β reduction

(<function> <argument>)

  1. applicative order β reduce <argument> to <argument value>
  2. applicative order β reduce <function> to <function value>
  3. if <function value> is λ<name>.<body> then replace all free occurrences of <name> in <body> with <argument value> and applicative order β reduce the new <body>
  4. if <function value> is not a function then return (<function value> <argument value>)

Evaluation termination

Not all expression evaluations terminate:
(λs.(s s) λs.(s s))

There is no way of telling whether or not the evaluation of an expression will ever terminate!

1st Church-Rosser theorem

If there are two distinct reductions or sequences of reductions that can be applied to the same term, then there exists a term that is reachable from both results, by applying (possibly empty) sequences of additional reductions (see picture)

2nd Church-Rosser theorem

If an expression has a normal form, then it may be reached by normal order evaluation
